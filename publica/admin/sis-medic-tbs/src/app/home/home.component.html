Sistema de Citas Medicas

Bienvenido a SIS Medic TBS un Sistema de Citas Médicas útil para consultorios médicos y/o médicos independientes.

Características:

  • Gestión de Citas
  • Gestión de Médicos
  • Gestión de Pacientes
  • Gestión de Usuarios con Acceso al Sistema
  • Creación de Usuarios
  • Historial de Citas por Paciente
  • Historial de Citas por Medico
  • Buscador avanzado por: Palabra clave, medico, paciente y fecha.
  • Almacenamiento de Exámenes y Radiografías: Ver, Modificar y Eliminar
  • Odontograma por paciente
